#  Perhitungan Sederhana
Proyek Tugas Pengujian Perangkat Lunak  
Kelas PHP sederhana untuk menghitung statistik dasar dari array angka, dilengkapi unit test menggunakan PHPUnit.


## Fitur Utama (`Average.php`)

| Method | Fungsi |
|--------|-------|
| `mean()` | Rata-rata (mean) |
| `median()` | Nilai tengah (median) |
| `sum()` | Jumlah total |
| `countElements()` | Jumlah elemen |
| `mode()` | Nilai paling sering muncul (bisa >1) |
| `minValue()` | Nilai terkecil |
| `maxValue()` | Nilai terbesar |

> Semua method **memvalidasi input**:  
> - Array **tidak boleh kosong**  
> - Semua elemen **harus numerik**
